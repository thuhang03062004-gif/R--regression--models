# Linear Regression & Statistical Diagnostics 📈

## 📖 Project Description
This repository contains a comprehensive case study on building, diagnosing, and visualizing linear regression models. As the **Team Leader**, I coordinated the project to apply advanced mathematical statistics to real-world data, evaluating the reliability and fit of statistical models.

## 🛠️ Technologies & Libraries
* **Language:** R
* **Key Libraries:** `plot3D`, `scatterplot3d`, `moments`, `EnvStats`

## ⚙️ Key Techniques & Features
* **Data Preprocessing:** Data cleaning and handling missing values.
* **Modeling:** Building Linear Regression models and conducting ANOVA.
* **Statistical Diagnostics:** Residual analysis, correlation analysis, and probability distributions.
* **Multi-dimensional Visualization:** Generating P-P plots, Q-Q plots, and 3D surface plots for accurate mathematical evaluation.

## 📂 Repository Structure
The project is divided into structured case study scripts:
* `CS_TH#1.R` - `CS_TH#4.R`: Step-by-step implementation from data exploration to advanced model diagnostics.

## 🏆 Key Results
* Successfully built and evaluated the reliability of mathematical models.
* Achieved an excellent score of **9.0/10.0** in the *Linear Statistical Methods* course.
